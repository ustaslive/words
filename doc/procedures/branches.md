# Stable

`master` branch is only for stable versions.

Only `develop` branch can be merged to `master`

# Development and testing

`develop` branch is for code in development. it will merge into `master` only when ready for new stable.
    all development and testing code is here.


# Working branches

Branches where new feature are developed.
Name them these ways
`<developer>/<feature_descr>`
`feature/<feature_descr>`

These branches are merged to `develop`
